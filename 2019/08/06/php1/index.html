PHP笔记（一）
发布于 2019-08-06   更新于 2019-08-06

1、什么是PHP？为什么要学PHP？
PHP是嵌入html页面中的脚本语言，是目前最流行的网站开发语言，在几乎所有的平台中都可以运行，很多的企业都在使用PHP作为开发语言。
2、PHP的基础规范
（1）PHP脚本以结束或者最后不加?>
（2）php脚本可以单独存在，也可以与html页面结合
（3）php文件扩展名规范，一般是.php结尾，但是，可以使用其他命名，例如.phtml等。
3、准备PHP的环境
phpstudy的安装使用：

安装好后，配置好，apache服务器的web服务器根目录，默认配置首页，配置日志记录，配置错误引导页面，配置php的扩展功能打开php_mysql.dll，再安装notepad++或者sublime text3.

4、php的输入输出

php的输出：echo print printf sprintf print_r die var_dump

例如 echo “1123”；

die 结束，不执行后面的；var_dump 可以显示数据类型；

echo(‘111
666’);
print (‘111
666’);
var_dump(‘222’);//**显示数据类型
die(‘**程序结束**’);

5、变量的数据类型

整型 （int）：整数；

布尔型(bool)：true/false；

字符串型(string)：“” ‘’ <<<EOF EOF

浮点型(float)：小数；

数组(array):[1,2,4] 键值对[‘name’ =>>’zs’,’age’=>> 24]

对象： new classname()

资源类型：打开的文件 打开的网络连接 STDIN STDOUT

备注：

php的变量必须以$符号开始，后面跟着变量的名称。

变量名必须与字母或者下划线字符开始，

变量名必须只能包含字母数字字符以及下划线（A-z、0-9和_）,

变量名不能包含空格，且区分大小写。

定义变量：使用php中的函数define()来定义常量，常量在使用时，不需要$符号。

6、常用的系统变量

FILE:php程序文件名/绝对路径
DIR:被调用的文件脚本所在的目录
FUNCTION:函数名
LINE:php程序文件行数
PHP_VERSION:当前解析器的版本号
PHP_OS:执行当前PHP版本的操作系统名称
_
7 单引号与双引号的区别
php中将任意字符（除了单引号）放入单引号中，没有意义
在双引号中放入特殊字符，就会有特殊字符。
EOF中变量有意义，其他的特殊字符都没有意义。
转义：\ 普通字符变为特殊字符,特殊字符转变为普通字符。
